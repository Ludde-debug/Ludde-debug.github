    • Vad är Cpu och vad den används till och hur den fungerar

    CPU fungerar på det sättet att den gör beräkningar och ger ut instruktioner till datorn, den här delen brukar även kallas för datorns hjärna. Processorn innehåller olika delar, så kallade små processorer och komponenter som sammanlagt utgör större delen av Cpu(Central Processing Unit) Processorn sitter fast på ett modekort(som jag kommer att prata om lite senare!), för att den ska kunna läsa av datan så måste den först kunna läsa av "adresser" ifrån de här adresserna finns data som sedan lagras/tas upp av Cpu.

    Cpu är även viktig eftersom den kommunicerar med andra delar i datorn/maskinen. Detta gör att den(processorn) kan köra alla program på datorn. Ett exempel på detta är när den samarbetar med minnet. Våra datorer kan bara genomföra de instruktioner som finns i minnet. Antingen är det i Ram-minnet(som jag senare kommer att prata om) detta är arbetsminnet. Det finns även i Rom-minnet. Där den endast läses av, där kan inte informationen ändras. Där oavsett om du stänger av datorn kommer det att lagras i Rom-minnet.
